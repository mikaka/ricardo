A super API for Ricardo

This is a placeholder page to show you the REST API. Use httpie to post JSON to the application.

To see all tasks, you can do a GET: 

	    http GET localhost:9000/tasks

To create new task, do a post 

	    http POST localhost:9000/tasks title="tache balaie" description="passer le balaie"

To modify a task, do a put 

	    http PUT localhost:9000/tasks/ title="tache netoyage" description="Some Body"

See all task recorded: /tasks
